# 📊 Business Sales Analysis Dashboard using Power BI

## Overview
This project showcases an interactive Business Sales Analysis Dashboard built in Microsoft Power BI. It provides insights into sales performance through KPIs, revenue trends, top-selling products, and regional analysis, enabling data-driven business decisions.

## Features
- 📈 Revenue Trend Analysis
- 🏆 Top 10 Products by Revenue
- 🌍 Country-wise Sales Analysis
- 📊 KPI Cards
  - Total Revenue
  - Total Orders
  - Total Customers
  - Total Products Sold
- 🎛 Interactive Filters (Year, Country, Product)

## Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX
- CSV Dataset

## Data Cleaning
- Removed duplicates
